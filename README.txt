MTA-turnstiles
===============

This is an application designed to visualize the MTA turnstile data
available at http://mta.info/developers/turnstile.html

Development
===========

To run, place a copy of tornado inside the top level directory
http://www.tornadoweb.org/

see data.txt for steps to load data

For information on improving this application create a fork of it
on github, or contact @jehiah
